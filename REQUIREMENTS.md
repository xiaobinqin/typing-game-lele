# 需求文档 — 打字大挑战·乐乐版

## 1. 项目背景

目标用户为小学 1—6 年级学生，旨在通过趣味性键盘打字游戏帮助学生：

- 巩固拼音（声母、韵母、整体认读音节、拼音音节）的记忆与书写
- 练习汉字与词语的拼音标注
- 培养键盘输入的基本操作习惯

内容与**人教版小学语文教材**同步，按年级分级，难度循序渐进。

## 3. 用户角色

| 角色 | 说明 |
|------|------|
| 小学生（主用户） | 1—6 年级，自主选择年级和内容进行练习 |
| 家长 / 教师（辅助） | 引导学生选择合适难度，查看练习记录 |

## 4. 功能需求

### 4.1 年级难度分级

| 等级 | 对应年级 | 主要内容 |
|------|----------|----------|
| 启蒙级 | 一年级 | 声母、韵母、整体认读 |
| 基础级 | 二年级 | 拼音音节、汉字拼音 |
| 进阶级 | 三—四年级 | 汉字拼音、词语 |
| 熟练级 | 五—六年级 | 词语综合练习 |

### 4.2 练习内容类型（6 类）

| 类型 | 分组 | 说明 |
|------|------|------|
| 声母 | 拼音基础 | 23 个声母拼音输入 |
| 韵母 | 拼音基础 | 单韵母、复韵母、鼻韵母 |
| 整体认读音节 | 拼读练习 | 16 个整体认读 |
| 拼音音节 | 拼读练习 | 两拼、三拼音节 |
| 汉字 | 字词学习 | 人教版生字拼音输入 |
| 词语 | 字词学习 | 人教版词语拼音输入 |

### 4.3 游戏模式（4 种）

#### 4.3.1 消消乐（Falling）
- 字/词从屏幕顶部随机落下，玩家在落到底部前输入正确拼音消除
- 支持**慢速 / 中速 / 快速**三档，进入前选择
- 答题数量增加后自动加速（难度递增）
- 错误不扣分，但字到达底部扣除一条生命（共 3 条）
- 失去所有生命后游戏结束，显示分数

#### 4.3.2 闯关模式（Challenge）
- 每关固定 20 道题，依次作答
- 正确率 ≥ 60% 视为通关，按正确率颁发 1—3 颗星
- 游戏结束后展示错题回顾
- 自动保存得分记录

#### 4.3.3 竞速模式（Speed）
- 60 秒内尽可能多答题
- 连续答对触发连击加分（最多 +10 分/题）
- 倒计时末 10 秒高亮警示
- 结束后可录入昵称，写入本地排行榜（Top 10）

#### 4.3.4 练习模式（Practice）
- 无时限、无生命限制，随意切换题目
- 支持"显示提示"功能（显示正确答案）
- 实时统计本轮正确率
- 适合课前预习、课后复习

### 4.4 页面流程

```
主菜单
  └─ 年级选择（4级）
       └─ 模式选择（4种模式 × 6类内容）
            ├─ 消消乐  →  速度选择  →  游戏  →  游戏结束
            ├─ 闯关模式  →  游戏  →  结果页
            ├─ 竞速模式  →  游戏  →  结果页
            └─ 练习模式  →  游戏
排行榜（竞速模式 Top 10）
```

### 4.5 记录与统计

- 每局游戏结果（模式、年级、内容类型、总题数、正确数、得分）本地 JSON 持久化
- 竞速模式支持昵称 + 分数排行榜，Top 10
- 返回各场景时恢复上次选中状态（年级、内容类型、游戏模式）

## 5. 非功能需求

| 类别 | 要求 |
|------|------|
| 性能 | 60 FPS 稳定运行，帧间绘制不超过 16ms |
| 跨平台 | 同一套代码支持 macOS / Windows / Linux |
| 字体 | 内置中文字体，不依赖系统预装字体 |
| 数据安全 | 存档文件写入用户主目录，打包后不污染应用包 |
| 可维护性 | 场景分文件，数据与代码分离（JSON 题库）|
| 界面风格 | 简洁蓝白风格，适合儿童，字体清晰，颜色明快 |

---

## 6. 题库数据来源

- `data/phonics.json`：声母、韵母、整体认读、拼音音节，按年级整理
- `data/characters.json`：人教版 1—6 年级生字及其拼音
- `data/words.json`：人教版 1—6 年级词语及其拼音
